abs() Ek number ka absolute value deta hai — matlab negative sign hata do. Internally ye number ka __abs__() dunder method call karta hai. Complex numbers pe bhi kaam karta hai (magnitude nikalke). 
# Gradient clipping in neural nets
grads = [-2.5, 0.8, -0.3, 3.1]
clipped = [max(-1, min(1, g)) if abs(g) > 1
           else g for g in grads]
print(clipped)
[-1, 0.8, -0.3, 1]
 ML/AI: Gradient clipping, distance calculation, financial P&L computation jahan sirf magnitude matter kare.
all() Iterable ke har element ko bool mein convert karta hai — ek bhi False mila toh turant False return karta hai (short-circuit). Generator pe kaam karta hai bina pura evaluate kiye. 
data = {"name": "Rahul", "age": 25,
         "email": "r@x.com"}
required = ["name", "age", "email", "phone"]
valid = all(data.get(k) for k in required)
print(valid)
False
 Form validation: Check karo ki API response mein saare required fields present hain ya nahi. Database insert se pehle constraints verify karna.
any() Iterable mein koi bhi ek True mila toh True return karta hai — bhi short-circuit. Ye basically OR operator ka iterable version hai. Empty iterable pe False deta hai. 
logs = ["INFO: started", "ERROR: timeout",
         "INFO: retry"]
has_error = any("ERROR" in line 
              for line in logs)
print(has_error)
True
 Log monitoring: Production logs mein error detect karna. Permission systems mein check karna ki user ke paas koi bhi required role hai ya nahi.
bool() Kisi bhi object ko True/False mein convert karta hai — object ka __bool__() call hota hai, ya agar woh nahi hai toh __len__(). 0, None, "", [], {} — sab False hain (falsy values). 
class Stack:
    def __init__(self):
        self.items = []
    def push(self, v): self.items.append(v)
    def __bool__(self):
        return bool(self.items)

s = Stack()
print(bool(s))
s.push(1)
print(bool(s))
False True
 Custom containers: Apne data structures mein truthiness define karna. Empty queue, stack, ya buffer ko if-condition mein directly use karna.
len() Object ka size/length return karta hai — internally __len__() call hota hai. O(1) hota hai most built-in types ke liye (list, dict, str) kyunki length cached hoti hai, recalculate nahi hoti. 
matrix = [[1,2,3],[4,5,6],[7,8,9]]

sum() Iterable ke elements ko left-to-right add karta hai, starting value default 0 hai. start parameter se lists/strings bhi concatenate kar sakte ho. Generator ke saath memory-efficient hai.

max() Iterable ka sabse bada element dhundta hai — internally comparison ke liye __gt__ use karta hai. key parameter se custom sorting logic laga sakte ho bina data change kiye.

min() Iterable ka sabse chota element dhundta hai. max() ka mirror image hai. Key function ke saath complex objects mein minimum dhundna easy ho jaata hai.

float() String ya int ko IEEE 754 double-precision float mein convert karta hai. "inf", "-inf", "nan" strings bhi accept karta hai. Precision issues hoti hain (0.1 + 0.2 ≠ 0.3).

str() Kisi bhi object ko human-readable string mein convert karta hai — object ka __str__() call hota hai. repr() se different hai — ye user ke liye hai, woh developer ke liye. Custom classes mein override kar sakte ho. 
class Money:
    def __init__(self, amt, cur):
        self.amt, self.cur = amt, cur
    def __str__(self):
        return f"{self.cur} {self.amt:,.2f}"

m = Money(125000, "₹")
print(str(m))

list() Kisi bhi iterable ko list mein convert karta hai — ek baar iterate karta hai aur sab elements ko list mein store karta hai. Generator ko exhaust kar deta hai. Shallow copy banaata hai.

tuple() Iterable ko immutable sequence mein convert karta hai. Tuples hashable hain (if elements bhi hashable hain), isliye dict keys ya set elements ban sakte hain. Memory mein list se thoda efficient hain.

ord() Character ko uska Unicode code point (integer) return karta hai. chr() ka inverse. String comparison internally yehi use karta hai. Password strength, text statistics ke liye useful. 
def pwd_strength(pwd):
    has_upper = any(65 <= ord(c) <= 90  for c in pwd)
    has_digit = any(48 <= ord(c) <= 57  for c in pwd)
    return "Strong" if has_upper and has_digit else "Weak"

print(pwd_strength("Pass123"))
Strong
 Security: Password validation, text encoding detection, sorting multilingual strings, checksum calculation.

next() Iterator se agla element return karta hai — __next__() call hota hai. StopIteration raise hoti hai agar koi element nahi. Default parameter se error avoid kar sakte ho — yeh ek powerful pattern hai.

    {"id":2, "active":True},

# First active user without loop
first = next(

type() Object ka class/type return karta hai. 3-argument form mein nayi class dynamically create karta hai (metaclass call) — yeh bahut powerful hai. isinstance() se better nahi — subclass check nahi karta.

Node 42

id() Object ka unique identity (memory address) return karta hai — CPython mein actual memory address hota hai. Is or is not operator isi id ko compare karta hai. Object lifetime tak id unique rehti hai.

c = a.copy()  # new object

isinstance() Object kisi class ka instance hai ya nahi — inheritance chain bhi check karta hai. Tuple of types pass kar sakte ho (OR condition). type() se better hai kyunki polymorphism respect karta hai. 
from numbers import Number
def safe_sqrt(x):
    if not isinstance(x, Number):
        raise TypeError(f"{x!r} not numeric")
    if x < 0:
        return complex(0, (-x)**0.5)
    return x**0.5
print(safe_sqrt(-4))
2j

callable() Object call kiya ja sakta hai ya nahi check karta hai — __call__ method hai ya nahi. Functions, classes, lambdas, aur __call__ wale objects sab callable hain. Runtime mein function existence check ke liye.

super() Parent class ka proxy object return karta hai — Method Resolution Order (MRO) follow karta hai. Multiple inheritance mein cooperative calling possible hai. Python 3 mein arguments optional hain (zero-arg super()).

getattr() Object se named attribute ka value programmatically lena. obj.name jaisa hai, lekin attribute ka naam runtime mein string se decide hota hai. Default value dene se AttributeError avoid hota hai. 
class Config:
    debug = True
    db_host = "localhost"

cfg = Config()

vals = {k: getattr(cfg, k, None) for k in keys}

hasattr() Object mein attribute exist karta hai ya nahi check karta hai — internally getattr() try karta hai aur exception pakad leta hai. Safe attribute access ke liye use karo. Duck typing pattern mein essential hai. 
def serialize(obj):
    if hasattr(obj, "to_dict"):
        return obj.to_dict()
    elif hasattr(obj, "__dict__"):
        return obj.__dict__
    return str(obj)

bytes() Immutable byte sequence banata hai (0-255 values). String se convert karne ke liye encoding specify karna zaroori hai. Network communication aur file I/O mein fundamental. Slice karo toh bhi bytes milte hain. 
# Simple XOR encryption
msg = bytes("Hello", "utf-8")
key = 0x42
enc = bytes(b ^ key for b in msg)
dec = bytes(b ^ key for b in enc)
print(enc.hex(), dec)
0a272d2d2d b'Hello'
 Networking/Crypto: TCP packets send karna, file binary read, JWT tokens, image data handle karna, protocol buffers.
bytearray() bytes() jaisa lekin mutable hai — in-place modify kar sakte ho. Jab bada binary data modify karna ho aur copy nahi banana toh use karo. Buffer protocol support karta hai. 
# In-place image channel swap BGR→RGB
pixel = bytearray([255, 128, 0])  # BGR
pixel[0], pixel[2] = pixel[2], pixel[0]
print(list(pixel))  # RGB
[0, 128, 255]
 Image processing: OpenCV BGR to RGB in-place convert, network buffer reuse, serial port communication, binary protocol implementation.
memoryview() Buffer object ka zero-copy view banata hai — data copy kiye bina large bytes/bytearray ke portions access kar sakte ho. Slicing pe copy nahi hoti — isliye performance-critical code mein use hota hai. 
data = bytearray(range(256))
mv   = memoryview(data)
# No copy — just a view of bytes 10-20
chunk = mv[10:15]
print(list(chunk))
print(chunk.nbytes)
[10, 11, 12, 13, 14] 5
 High-performance I/O: Large video/audio buffers process karna bina copy ke, socket data zero-copy parsing, scientific computing (NumPy interface).
frozenset() set() jaisa lekin immutable aur hashable hai — dict keys aur set elements ban sakta hai. Set operations (union, intersection) support karta hai. Constant lookup tables ke liye ideal. 
RESERVED_WORDS = frozenset([
    "admin", "root", "system", "api"
])
username_cache = {}
def register(name):
    if name in RESERVED_WORDS:
        return "Reserved!"
    return "OK"
print(register("admin"))
Reserved!
 Constant sets: Reserved keywords list, banned words, whitelist/blacklist jo change nahi honi chahiye. Dict key ke roop mein permission sets store karna.
complex() Real + imaginary parts se complex number banata hai (a + bj form). Python mein natively supported hai. Engineering signals, FFT, electrical circuits ke calculations ke liye. abs() magnitude deta hai complex numbers ki. 
import cmath
# AC circuit impedance calculation
R = complex(50, 0)   # Resistance
XL = complex(0, 30)  # Inductive
Z = R + XL
print(Z, abs(Z).__round__(2))
(50+30j) 58.31
 Signal processing: FFT calculations, AC circuit analysis, quantum computing simulations, 2D rotation/transformation in graphics.

globals() Current module ka global symbol table dict return karta hai — live reference hai, copy nahi. Isme changes actual globals mein reflect hoti hain. Module-level variables, functions sab isme hain.

locals() Current function ka local namespace dict return karta hai. Outer scope mein changes reflect nahi hote (unlike globals). Debugging, string formatting, template rendering mein useful. Function ke bahar globals() jaisa hota hai. 
def make_report(name, score, rank):
    template = (
        "Student: {name}\n"
        "Score: {score}\nRank: {rank}"
    )
    return template.format(**locals())

print(make_report("Anika", 92, 3))
Student: Anika Score: 92 Rank: 3
 Template rendering: Local variables directly template mein inject karna bina manually dict banaye. Debug snapshots, SQL query builders.
vars() Object ka __dict__ attribute return karta hai — instance variables ka dict. Bina argument ke locals() jaisa. Custom objects ko dict mein convert karne ka clean tarika. Serialization mein very useful. 
from dataclasses import dataclass

@dataclass
class Product:
    name: str
    price: float
    stock: int

p = Product("Laptop", 45999, 10)
print(vars(p))
{'name': 'Laptop', 'price': 45999, 'stock': 10}
 Serialization/APIs: Dataclass/model instances ko JSON-serializable dict mein convert karna. ORM objects ko API response dict mein map karna.
repr() Object ka unambiguous string representation return karta hai — developer ke liye (str() user ke liye hota hai). Ideally iska output Python mein paste karke same object recreate ho jaaye. Debugging mein essential. 
class Point:
    def __init__(self, x, y):
        self.x, self.y = x, y
    def __repr__(self):
        return f"Point({self.x!r}, {self.y!r})"

pts = [Point(1,2), Point(3,4)]
print(repr(pts))
[Point(1, 2), Point(3, 4)]
 Debugging/Logging: Object state clearly log karna, REPL mein informative output, test failures mein exact values dikhana.

slice() Slice object banata hai (start, stop, step) — reusable aur named slice define karne ke liye. a[1:5:2] internally slice(1,5,2) ban jaata hai. Custom __getitem__ mein slice objects receive kar sakte ho.
